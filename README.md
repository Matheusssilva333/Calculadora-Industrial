Calculadora Industrial

Objetivo: O objetivo dessa calculadora é realizar cálculos mecânicos além das operações básicas.

Funcionalidades: 
- Soma
- Subtração
- Divisão
- Multiplicação
- Cálculo de RPM
- Cálculo de Torque
- Cálculo de Potência

Observação: O Cálculo de RPM da Calculadora é para motores elétricos.
